# Lofi Digital Clock

A chill, aesthetic digital clock and calendar for your workspace. Features smooth gradients, floating particles, and a relaxing lofi vibe.

## Features

- **Aesthetic Visuals**: Beautiful gradient backgrounds with animated particle effects.
- **Lofi & Anime Themes**: Switch between curated themes to match your mood.
- **Split Layout**: Clean interface with a large digital clock and a monthly calendar.
- **Responsive Design**: Looks great on desktops, tablets, and mobile devices.
- **Customizable**: Toggle between 12h and 24h formats.

## Tech Stack

- **React**: UI Library
- **TypeScript**: Type safety
- **Vite**: Fast build tool
- **Tailwind CSS**: Styling
- **DaisyUI**: UI Components
- **Lucide React**: Icons

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/abhijit-the-ghost/lofi-digital-clock.git
    ```
2.  **Install dependencies**
    ```bash
    yarn install
    ```
3.  **Run development server**
    ```bash
    yarn dev
    ```

## Deployment

This project is configured to automatically deploy to GitHub Pages using GitHub Actions.

1.  Push your changes to the `main` branch.
2.  The `Deploy to GitHub Pages` workflow will run automatically.
3.  Your site will be live at `https://<your-username>.github.io/lofi-digital-clock/`.
